

## Description

An real time project management application using NestJs and typeORM

## Project setup
configure your .env file
```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ nest start

# watch mode
$ nest start --watch

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
